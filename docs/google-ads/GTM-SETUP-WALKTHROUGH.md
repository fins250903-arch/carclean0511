## 【STEP 1】Google広告：コンバージョンアクションの作成

### 1-1. 管理画面を開く

1. https://ads.google.com/ にログイン
2. 左メニュー **「目標」** → **「コンバージョン」** → **「概要」**

### 1-2. 電話コンバージョンを作成

1. **「+ コンバージョンアクションを作成」** をクリック
2. **「ウェブサイト」** を選択
3. 次のように入力：

| 項目 | 入力値 |
|---|---|
| カテゴリ | 連絡先（なければ「その他」） |
| コンバージョン名 | `電話問い合わせ_CV` |
| 値 | コンバージョンに値を使用しない |
| カウント | 1回 |
| クリック後のコンバージョン期間 | 30日 |
| ビュースルーコンバージョン期間 | 1日 |

4. **「タグの設定」** で **「Googleタグマネージャーを使用する」** を選択
5. 表示される以下をメモ（後でGTMに入力）：

```
コンバージョンID: AW-XXXXXXXXXX
コンバージョンラベル（電話）: xxxxxxxxxxxxx
```

### 1-3. LINEコンバージョンを作成

同様の手順で、もう1つ作成：

| 項目 | 入力値 |
|---|---|
| コンバージョン名 | `LINE問い合わせ_CV` |
| カテゴリ | その他 |

LINE用の **コンバージョンラベル** も別途メモ（電話とは異なります）。

### 1-4. メモ用テンプレート

以下をコピーして値を埋め、作業時に手元に置いてください：

```
コンバージョンID（共通）: AW-
電話コンバージョンラベル:
LINEコンバージョンラベル:
```

> **エージェントへの共有:** 上記3つが分かれば、GTMタグの最終設定を確認できます。

## 【STEP 3】GTM：タグ・トリガーの設定

GTM: https://tagmanager.google.com/ → コンテナ **GTM-WLNM4GWN**

### 3-1. 組み込み変数を有効化

1. 左メニュー **「変数」**
2. **「設定」**（組み込み変数）をクリック
3. **「クリック」** にチェック：
   - Click URL
   - Click Element
   - Page URL
4. **「保存」**

### 3-2. コンバージョン リンカー（全ページ・1つだけ）

1. **「タグ」→「新規」**
2. タグ名: `Google広告 - コンバージョンリンカー`
3. タグの種類: **コンバージョン リンカー**
4. コンバージョンID: `AW-XXXXXXXXXX`（STEP1で取得）
5. トリガー: **All Pages**
6. 保存

### 3-3. トリガー：電話クリック

1. **「トリガー」→「新規」**
2. 名前: `TR - 電話クリック`
3. 種類: **クリック - すべての要素**
4. 発生場所: **一部のクリック**
5. 条件（2行追加・AND）:

| 変数 | 演算子 | 値 |
|---|---|---|
| Click URL | 正規表現に一致 | `^tel:` |
| Click URL | 含む | `07084280866` |

※ ハイフン付き `tel:070-8428-0866` にも対応するため、2行目は `0708428` だけでも可。

**代替（カスタムイベント）:** サイトが `phone_click` を dataLayer に送るため、以下でも可：

- 種類: **カスタムイベント**
- イベント名: `phone_click`

### 3-4. トリガー：LINEクリック

1. **「トリガー」→「新規」**
2. 名前: `TR - LINEクリック`
3. 種類: **クリック - すべての要素**
4. 発生場所: **一部のクリック**
5. 条件:

| 変数 | 演算子 | 値 |
|---|---|---|
| Click URL | 含む | `lin.ee/Xs8Orp2` |

**代替:** カスタムイベント `line_click`

### 3-5. タグ：電話コンバージョン

1. **「タグ」→「新規」**
2. 名前: `Google広告 - CV 電話問い合わせ`
3. 種類: **Google 広告 コンバージョン トラッキング**
4. 設定:

| 項目 | 値 |
|---|---|
| コンバージョンID | `AW-XXXXXXXXXX` |
| コンバージョンラベル | 電話用ラベル（STEP1） |
| コンバージョンの値 | （空欄） |
| 通貨コード | JPY |

5. トリガー: `TR - 電話クリック` または `phone_click` カスタムイベント
6. 保存

### 3-6. タグ：LINEコンバージョン

1. 名前: `Google広告 - CV LINE問い合わせ`
2. コンバージョンラベル: **LINE用**（電話と取り違えない）
3. トリガー: `TR - LINEクリック` または `line_click`
4. 保存

### 3-7. 二重計測の防止

次の組み合わせは **どちらか一方だけ** にしてください。

| パターン | 説明 |
|---|---|
| A（推奨） | カスタムイベント `phone_click` / `line_click` のみ |
| B | クリックURL（`tel:` / `lin.ee`）のみ |

**両方付けると1クリックで2CVになる** ため注意。

本サイトは dataLayer イベントを送るため、**パターンA（カスタムイベント）推奨**。

## トラブルシューティング

| 症状 | 確認すること |
|---|---|
| プレビューでイベントが出ない | デプロイ済みか / 広告ブロックOFFか |
| イベントは出るがCVタグが発火しない | トリガー名とイベント名の一致 |
| CVタグは出るが広告に記録されない | ID・ラベルの誤り / 24時間待つ |
| 固定バーだけ計測される | 旧デプロイのまま → 最新をデプロイ |
| CVが2倍になる | クリックURLトリガーとカスタムイベントの二重設定 |
