## PART A：Google広告（今すぐやること）

### A-1. 古いコンバージョンを整理する

1. https://ads.google.com/ → **目標** → **コンバージョン** → **概要**
2. 以前作った「動いていない」アクションを1件ずつ開く
3. 画面下部 **「コンバージョンアクションを削除」**（または「アーカイブ」）
4. 電話・LINE用に **新しく作り直す**（質問3で B を選択済み）

> 削除できない（Primary設定など）場合は、名前の末尾に `_旧` を付けて **セカンダリ** に変更し、新規2件を Primary として作成してください。

---

### A-2. 電話コンバージョンを新規作成

1. **「+ コンバージョンアクションを作成」**
2. **「ウェブサイト」** をクリック
3. 次の表どおり入力：

| 画面の項目 | 入力する値 |
|---|---|
| カテゴリ | **連絡先**（なければ「その他」） |
| コンバージョン名 | `電話問い合わせ_CV` |
| 値 | **コンバージョンに値を使用しない** にチェック |
| カウント | **1回** |
| クリック後のコンバージョン期間 | **30日** |
| ビュースルーコンバージョン期間 | **1日** |
| 主な操作 | **Primary（主要な操作）** にする |

4. **「完了」** または **「タグの設定」** へ進む
5. **「Googleタグマネージャーを使用する」** を選択
6. 表示される **コンバージョンラベル** をコピー（英数字のみ、例: `kAbCdeFGhIjKlMnO`）
7. このシート上部の「電話問い合わせ_CV」欄に貼り付け

**確認:** コンバージョンID は `AW-17667339500` と表示されていること（アカウント共通）。

## よくあるつまずき

| 症状 | 対処 |
|---|---|
| ラベルが見つからない | コンバージョン詳細 → タグの設定 → GTM を再表示 |
| プレビューでイベントが出ない | サイトのデプロイ完了を待つ / 広告ブロックOFF |
| CVタグが Not Fired | トリガー名 `phone_click` のスペル確認 |
| CVが2倍 | クリックURLトリガーとカスタムイベントの **重複を削除** |
| 広告に24時間反映されない | 正常。プレビューでタグ発火を先に確認 |
